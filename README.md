## Project Background
- ### 필요성
  - 국내 스마트축산·스마트팜 보급이 빨라지면서 축사 내부를 항상 보이는 상태로 유지하는 영상 모니터링이 필수인데, 실제 돈사·우사처럼 분진·습기·분뇨가 공존하는 환경에서는 CCTV/AI 카메라 앞 투명창에 오염물이 짧은 주기로 들러붙어 영상이 흐려지는 문제가 반복된다. 이때 영상이 흐려지면 그 위에서 동작하는 이상행동 탐지, 개체 모니터링 같은 AI 서비스도 동시에 성능이 떨어져서 “장비는 설치했는데 활용은 잘 안 되는” 현장 불만으로 이어진다. 이 문제를 줄이려면 카메라 자체보다 렌즈 오염을 지연·차단해 주는 전용 하우징이 먼저 필요하다.
  - 현장에서는 사람이 사다리를 타고 올라가 닦거나, 먼지가 덜한 위치로 카메라를 옮기는 식의 수동 조치를 해 왔는데, 축사 구조상 고정 설치가 많고 바닥이 젖어 있거나 미끄러운 경우도 많아서 “정기 청소를 전제로 한 모니터링”은 지속가능하지 않다. 따라서 현장의 오염 양상을 반영한 분진 방지 구조를 미리 설계·적용해 유지보수 주기를 뒤로 미루는 산학 협력이 필요하다.
- ### 기존 해결책의 문제점
  - 단순 투명 커버·보호 케이스는 축산 환경의 분진 방향성(위·측면 유입), 습기와 분진의 결합, 거미줄·미세분진에 의한 장기부착을 고려하지 않아 일정 기간이 지나면 결국 같은 위치에 먼지가 쌓여 화면이 뿌옇게 된다. 다시 말해 “씌우기만 하는” 방식은 축산용으로는 한계가 있다.
  - 영상 센서를 더 비싼 장비로 바꾸거나, 정밀한 광학·LiDAR형 보호 장치를 쓰는 방법도 있으나 비용이 높고 환경 변화에 민감해서 축산업에서 널리 쓰기 어렵다. 또 기종별 설치 여건이 제각각이라 표준화가 안 되는 문제가 있어, 카메라 본체는 그대로 두고 앞단 결합부만 교체할 수 있는 모듈형 하우징이 요구된다.
  
## System Design
  - ### System Requirements
    - 분진 차단 + 공기 흐름 형성: 렌즈 전면으로 일정한 공기 흐름을 보내 분진이 직접 안착하지 못하게 하고, 위·옆에서 들어오는 먼지도 앞쪽 형상으로 꺾어 주는 구조여야 한다. 이를 위해 팬(선풍기형/서큘레이터형)과 통로 형상이 설계 요구사항에 포함된다.
    - 현장 적응형·교체형 구조: 돈사에서 실제로 쓰는 여러 기종의 CCTV/AI 카메라에 공통으로 쓸 수 있도록 앞단 어댑터를 교체형으로 설계하고, 나중에 3D 프린터로 크기만 조정해 재출력할 수 있어야 한다. 이렇게 해야 설치 환경이 다른 농가에도 빠르게 보급된다.
    - 밀폐성·내구성 확보: 하우징 자체는 축산 환경의 습기·분뇨에 장기간 노출되므로 소재 선정, 결합부 밀폐, 팬 고정 방식에 대한 내구성 평가가 요구된다. 또 장기 운용 시 공기 흐름이 균일하게 유지되는지를 검증해야 한다.
    - 유지보수 단순화: 완전 무청소를 목표로 하기보다 “청소 주기를 최대한 늦추는 구조”로 설계하고, 실제 청소가 필요할 때는 간단히 분리해 닦을 수 있어야 한다. 이는 현장 피드백을 반영한 결과 보고서의 설계 방향과도 일치한다.
    
## Case Study
  - ### Description
  - 오염 실험
    : 기존 설치 카메라 앞에 개발한 분진 방지 하우징을 장착한 뒤 일정 기간 영상 품질 변화를 비교했다. 이때 기준 영상은 (A)오염도 정상, (B)약한 오염도, (C)심각한 오염도 단계로 나누어 기록했다. 하우징 적용군에서는 (B)→(C)로 넘어가는 시간이 지연되었고, 관리자가 닦는 주기도 뒤로 밀렸다.
  - 구조 실험
    : 3D 모델링으로 팬이 렌즈 정면으로 바람을 보내는 선풍기 방식과, 공기를 순환시켜 전면부에 난류를 만들어주는 서큘레이터 방식을 모두 설계해 풍량을 비교했고, 축사처럼 먼지가 위·옆에서 떨어지는 환경에서는 전면 유속이 확보되는 구조가 더 유리하다는 것을 확인했다. 이 결과는 이후 설계 개선 항목에 반영되었다.
  - 모듈·교체성 검증
    : 결과 보고서에서는 앞단 결합부를 교체형으로 만들어, 렌즈 높이나 본체 너비가 다른 카메라에도 동일 하우징을 쓸 수 있다는 점을 명시했다. 이 결과는 제안 단계에서 설정했던 ‘대학–기업 간 역할 구분’이 실제 수행 과정에서도 자연스럽게 작동했음을 보여준다.
  
## Conclusion
  - ### 분진 환경에서도 “사용 가능한 영상”을 확보하는 하드웨어 전처리 장치의 필요성을 검증
    : 이번 프로젝트는 스마트축산에서 흔히 간과되는 “카메라 앞단의 물리적 오염”을 문제 정의 단계에서부터 명확히 했고, 이를 지연·차단하는 하우징을 실제 형상으로 제시함으로써 소프트웨어·AI 개선만으로는 해결되지 않는 현장의 병목을 줄일 수 있음을 보여줬다. 결과적으로 영상 선명도가 유지되면 그 위에서 돌아가는 행동 인식·이상징후 탐지의 인식률 저하 문제도 동반 개선될 수 있음을 확인했다.
  - ### 현장 실증이 가능한 모듈형 3D 설계 파이프라인을 마련
    : 설계–3D 모델링–축사 테스트–피드백 반영이라는 짧은 고리를 산학 협력 구조 안에 넣어, 다른 농가나 다른 카메라 기종에서도 크기와 결합부만 바꿔 재사용할 수 있는 형태를 만들었다. 이건 프로젝트 기획서에서 제시한 “하우징 내부 밀폐성 최적화 및 현장 설치 적합성 검토”라는 목표를 실제 결과물로 연결했다는 점에서 의의가 있다.
